MicroSD Card Setup (Windows Guide)

For the sake of keeping this guide simple, the steps are the exactly the same for people with MicroSD Cards smaller than 32 gigabytes.

  1. Download FAT32Format and run the program.
  2. Plug in your MicroSD Card into your computer. Make sure that it is not an empty partition. If it is, format it using Disk Management to exFAT or any other file format.
  3. Select your MicroSD Card in the "Drive" menu, and then select the allocation size to "32768".
  4. Press the "Start" button and wait until the process is complete.
  5. You can now eject the MicroSD Card, and move on to the next page.
