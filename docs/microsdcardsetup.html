MicroSD Card Setup (Windows Guide)

For the sake of keeping this guide simple, the steps are exactly the same for people with MicroSD Cards smaller than 32 gigabytes.

  1. Download FAT32Format and run the program.
  2. Plug in your MicroSD Card into your computer. Make sure that it is not an empty partition. If it is, format it using Disk Management to exFAT or any other file format.
  3. Select your MicroSD Card in the "Drive" menu, and then select the allocation size to "32768".
  4. Press the "Start" button and wait until the process is complete.
  5. You can now eject the MicroSD Card, and move on to the next page.

For a Linux guide, please look below.

MicroSD Card Setup (Linux Guide using GParted)

This guide is for MicroSD cards under 32GB.

For the further sake of simplicity, a GUI tool is intentionally used. You may also want to back up any important files on your MicroSD card before formatting.
  1. As the Windows guide specifies, plug your MicroSD card into your computer.
  2. Update your Linux system with sudo apt update, sudo pacman -Syu, or sudo yum update.
  3. Install GParted with sudo apt install gparted, sudo pacman -Sy gparted, or sudo yum install gparted, and run it with root privileges with sudo gparted.
  4. Select your device in the dropdown located at the top right corner of the GParted home screen.
  5. Right-click the partition, select "Format to", then select "fat32".
  6. Click the green check mark to apply all operations, then "Apply" at the warning dialog. Let it complete.
  7. Click "Close" when the operation is completed. Now you're free to move onto the next page!

For a macOS guide, please look below.

MicroSD Card Setup (MacOS Guide Using Disk Utility)

This works for 32GB MicroSD cards, and possibly >32GB MicroSD cards.

  1. Insert your MicroSD card into your MacOS device.
  2. Open Disk Utility, which is preinstalled on MacOS.
  3. Select the MicroSD card, and click on "Erase" in the top toolbar. Make sure it's the correct MicroSD card!
  4. Give the MicroSD card a name, and format it for "MS-DOS (FAT)"; otherwise, "ExFAT" may be sufficient.
  5. To securely format the MicroSD card, open "Security Options" and adjust the slider to your preference.
  6. Click "Erase" to format your MicroSD card.
